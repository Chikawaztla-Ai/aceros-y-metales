01 — Alcance

Qué incluye el proyecto

Un sitio web completo orientado a generar ventas, no solo presencia. Cada elemento está pensado para convertir visitantes en clientes que soliciten cotización o compren directamente.

✓ Diseño UI personalizado (no template genérico)
✓ Catálogo de productos con fichas técnicas
✓ Formulario de solicitud de cotización
✓ Botón WhatsApp con mensaje predefinido
✓ Sección "Quiénes somos" + certif. / clientes
✓ Tienda en línea (WooCommerce o similar)
✓ Pasarela de pago configurada (Stripe / Conekta)
✓ Blog con estructura SEO desde el inicio
✓ Optimización SEO on-page (5 páginas clave)
✓ Alta en Google Search Console + Analytics
✓ Perfil de Google Business configurado
✓ Diseño responsive (móvil + escritorio)
✓ Velocidad optimizada (Core Web Vitals)
✓ SSL (HTTPS) + seguridad básica
✓ Capacitación básica al cliente (1 sesión)
✓ 30 días de soporte post-lanzamiento

02

Sem 3–6
04

Sem 6–7
05

Revisión final, correcciones, lanzamiento en producción, capacitación al cliente y activación del soporte post-lanzamiento de 30 días.
