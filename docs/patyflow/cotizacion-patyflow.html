01 — Alcance

Qué incluye el proyecto

Sitio completo para vender cursos grabados, clases en vivo y membresías recurrentes — con cobro automático y experiencia profesional para las alumnas.

✓ Diseño UI personalizado (identidad cálida, no plantilla)
✓ Plugin LMS (cursos grabados + clases en vivo)
✓ Sistema de membresías con pago recurrente
✓ Tienda en línea (WooCommerce)
✓ Pasarela: Stripe (recurrente) + Conekta (OXXO/SPEI)
✓ Área de alumna: Mi cuenta / Mis clases / Mi progreso
✓ Calendario de clases en vivo con inscripción
✓ Página "Sobre mí" con contenido real (certificaciones)
✓ Catálogo de clases/cursos (carga de hasta 15 clases)
✓ Blog con estructura SEO desde el inicio
✓ SEO on-page (5 páginas clave)
✓ Alta en Google Search Console + Analytics
✓ Google Business Profile configurado
✓ Diseño responsive (móvil + escritorio)
✓ SSL (HTTPS) + seguridad básica
✓ Capacitación + 30 días de soporte
