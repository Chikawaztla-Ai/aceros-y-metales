DecisiГіn: ВҝPor quГ© no un backend separado (Express, Nest)?

Next.js API Routes son suficientes para el MVP вҖ” cada ruta es una funciГіn serverless en Vercel, escala automГЎticamente y no requiere servidor dedicado. Si en V3 la complejidad del backend crece (agentes IA, ERP), se puede extraer a un servicio separado sin tocar el frontend.

Seguridad de API Routes

Webhook Mercado Pago

Validar x-signature header con HMAC SHA-256 usando el secret de MP. Rechazar cualquier request sin firma vГЎlida. Loggear intentos fallidos.

Rate Limiting

Limitar /api/cotizaciones a 5 req/min por IP (prevenir spam). Limitar /api/buscar a 20 req/min por IP (controlar costos OpenAI).

ValidaciГіn de inputs

Zod schemas en TODAS las API routes. Nunca confiar en datos del cliente. Sanitizar HTML en campos de texto libre (notes, company).

CORS y headers

Next.js maneja CORS por defecto (same-origin). Headers de seguridad en next.config.js: X-Frame-Options, CSP, HSTS.

05 вҖ” Calculadora de Peso

LГіgica, fГіrmulas y densidades

Componente reutilizable en homepage, catГЎlogo, producto y carrito. CГіdigo compartido via hook.

// lib/weight-calculator.ts const DENSITIES: Record<string, number> = { 'acero': 7850, // kg/mВі 'inox': 7900, 'aluminio': 2700, 'cobre': 8960, 'bronce': 8800, 'laton': 8500, 'nylamid': 1150, 'acetal': 1410, 'ptfe': 2200, }; type Profile = 'redondo' | 'cuadrado' | 'solera' | 'placa' | 'hexagonal' | 'tubo'; function calcWeight( material: string, profile: Profile, dims: { d?: number; a?: number; b?: number; wall?: number }, length: number, // metros qty: number ): number { const ПҒ = DENSITIES[material]; switch (profile) { case 'redondo': // ПҖ Г— (d/2000)ВІ Г— L Г— ПҒ Г— qty return Math.PI * (dims.d! / 2000) ** 2 * length * ПҒ * qty; case 'cuadrado': // (a/1000)ВІ Г— L Г— ПҒ Г— qty return (dims.a! / 1000) ** 2 * length * ПҒ * qty; case 'solera': case 'placa': // (a/1000) Г— (b/1000) Г— L Г— ПҒ Г— qty return (dims.a! / 1000) * (dims.b! / 1000) * length * ПҒ * qty; case 'hexagonal': // (вҲҡ3/2) Г— (a/1000)ВІ Г— L Г— ПҒ Г— qty return (Math.sqrt(3) / 2) * (dims.a! / 1000) ** 2 * length * ПҒ * qty; case 'tubo': // ПҖ Г— ((D/2000)ВІ - ((D-2t)/2000)ВІ) Г— L Г— ПҒ Г— qty const R = dims.d! / 2000; const r = (dims.d! - 2 * dims.wall!) / 2000; return Math.PI * (R ** 2 - r ** 2) * length * ПҒ * qty; } }

Nota para el sitio: La calculadora usa los perfiles internamente para el cГЎlculo matemГЎtico, pero por requerimiento del cliente, las presentaciones (Barra redonda, Solera, Placa, etc.) NO se muestran en la UI del catГЎlogo, filtros ni fichas de producto. El usuario selecciona el perfil solo dentro de la calculadora de peso.
