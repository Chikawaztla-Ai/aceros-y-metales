Lo mínimo para salir a producción

Si el objetivo es lanzar lo antes posible, el camino crítico es:

  1. Logo vectorial + fotos y datos de contacto.
  2. Catálogo real (Excel con productos y precios).
  3. Mercado Pago (credenciales) + datos bancarios.
  4. Correo (dominio verificado + correo del administrador).
  5. Supabase + dominio + deploy (lo montamos nosotros con tu OK).

Analítica y blog pueden quedar para después del lanzamiento sin bloquear la venta en línea.

Cualquier duda sobre un punto, la aclaramos antes de que lo consigas para no perder tiempo. Marca cada casilla conforme lo entregues.
